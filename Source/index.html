Steps to create GitHub Repository

  1. Create an account in GitHub and Login
  2. In the upper right corner of page, click on + and select New repository
  3. Type name of your repository ex:WebAndMobileProgramming
  4. Select public reposity between public and private repository options
  5. Select Initialize this repository with a README
  6. Click Create repository

Your repository has been created!!!

Open repository and copy link

Steps to Clone it to local repository

  1. Open GitHub Desktop Application
  2. Go to File ->Clone repository and paste the link copied from remote GitHub
  3. Clone the repository
  4. Create 2 directories
    • Source
    • Documentation
